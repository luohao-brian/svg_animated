# NLP Embedding 动画设计方案 (MBTI 类比版)

## 1. 概览
- **时长**: 约 90 秒
- **风格**: 趣味数据可视化 (Infographic Style), 扁平化深色主题
- **背景色**: `#0f172a` (Slate 900)
- **核心类比**: **词语 Embedding = 词语的“MBTI测试”**

## 2. 剧本与分镜 (Script & Storyboard)

### 第一幕: 人的“数字化” - MBTI 测试 (Digitizing Personality)
- **时间**: 00:00 - 00:15
- **场景**: 四分屏布局
  - 左上: 卡通人物 "Jay"
  - 右上: MBTI 问卷面板
  - 左下: 向量数值卡片 `Vec = [...]`
  - 右下: 雷达图
- **动作**: 
    1. Jay 进行 MBTI 测试，依次选择 E, N, T, J。
    2. 选项高亮，同时雷达图动态生长，展示四个维度的数值。
    3. 向量卡片同步显示对应的数值 `[0.9, 0.8, 0.6, 0.9]`。
    4. 所有元素保留在屏幕上，形成完整的人物画像。
- **字幕**: "我们可以用一组数字，精准描述一个人的性格 (MBTI)。"

### 第1.5幕: 余弦相似度原理 (Cosine Similarity Demo)
- **时间**: 00:15 - 00:25
- **场景**: 
  - 左侧: 交互式公式面板 `A · B / |A|×|B|`
  - 右侧: 单位圆与两个向量 A, B
- **动作**: 
    1. 向量 A 固定，向量 B 从 60度 旋转至 10度 (相似度变高)。
    2. 向量 B 继续旋转至 150度 (相似度变负)。
    3. 公式面板实时数值跳动，直观展示夹角与相似度的关系。
- **字幕**: "如何比较两个向量？我们需要“余弦相似度”。"

### 第二幕: 寻找“灵魂伴侣” (Finding Soulmates)
- **时间**: 00:25 - 00:40
- **场景**: 
  - 左侧: 坐标系，原点调整，展示向量。
  - 右侧: 匹配结果卡片。
- **动作**: 
    1. Jay 的性格向量出现。
    2. **Person A** (Soulmate) 向量出现，方向与 Jay 极近。
       - Label 位于向量下方，无遮挡。
       - 右侧显示 "Soulmate? 🤝", 相似度高 (e.g., 0.96)。
    3. **Person B** (Opposite) 向量出现，方向几乎相反。
       - Label 位于左侧，清晰可见。
       - 右侧显示 "Opposite? ❌", 相似度低 (e.g., -0.10)。
- **字幕**: "回到性格向量，计算夹角，就能找到 Soulmate。"

### 第三幕: 词语也有“性格” (Words Have Personalities)
- **时间**: 00:40 - 00:55
- **场景**: 三张单词卡片横向排列 (King, Man, Apple)。
- **动作**: 
    1. 单词卡片翻转/出现。
    2. 依次展示单词的“性格测试”维度：
       - "人类?", "男性?", "皇室?", "有生命?" 等。
    3. 每个维度打钩或打叉，对应形成潜在的特征向量。
- **字幕**: "Embedding 就是给单词做 1000 多道题的‘性格测试’。"

### 第四幕: 语义空间聚类 (Semantic Clustering)
- **时间**: 00:55 - 01:10
- **场景**: 巨大的二维语义空间。
- **动作**: 
    1. 背景浮现海量微小的词向量点 (Background Dots)，营造浩瀚星空感。
    2. 高亮几组特定的聚类：
       - **动物**: Cat, Dog, Tiger, Lion (绿色)
       - **水果**: Apple, Banana (黄色)
       - **动作**: Run, Jump, Walk (粉色)
    3. 相同类别的词向量方向基本一致。
- **字幕**: "在这个坐标系里，意思相近的词，向量方向几乎一致。"

### 第五幕: 著名的“国王与王后” (The King & Queen)
- **时间**: 01:10 - 01:30
- **场景**: 聚焦 King, Man, Woman, Queen 四个向量。
- **动作**: 
    1. 显示 **Man** 和 **King** 向量。
    2. 绘制差值向量 `+ Royal` (紫色)，连接 Man -> King。
    3. 显示 **Woman** 向量。
    4. 将 `+ Royal` 向量平移至 Woman 的末端。
    5. 动画演示向量相加，最终指向的位置浮现出 **Queen**。
- **字幕序列**: 
    1. "词向量的神奇之处：它们可以进行计算。"
    2. "第一步：King - Man = '皇室' 向量"
    3. "第二步：找到 Woman 向量"
    4. "第三步：Woman + '皇室' ..."
    5. "结果：Woman + '皇室' = Queen"
    6. "最终公式：King - Man + Woman = Queen"

## 3. 视觉设计重点
- **布局优化**: Scene 1 采用四象限布局，充分利用屏幕空间，避免字幕遮挡。
- **防遮挡**: Scene 2 标签位置精细调整 (Person A 下移, Person B 左移)。
- **清晰度**: 关键数据 (Vector, MBTI) 使用大字号或 Monospace 字体，雷达图数值与标签分离。
