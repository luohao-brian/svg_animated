# MNIST 神经网络训练实验指南 (45秒深度增强版)

## 1. 项目定位
本项目通过 45 秒的高保真动画，详尽展示深度神经网络（Fully Connected Network）学习 MNIST 手写数字的工业级过程。核心目标是通过强视觉冲击力的“信号流”和“参数回传”，解释前向传播（FWD）、反向传播（BWD）及梯度下降的数学美感。

## 2. 技术规格
- **框架**: Vue 3 (Composition API)
- **动画**: Anime.js (主控 Timeline 与数值补间)
- **视觉**: SVG + CSS 滤镜 (Glow/Drop Shadow)
- **时长**: 45,000ms (精准对齐)
- **录制**: 1080p, 60fps, 输出至 `video_records/mnist_demo.mp4`

## 3. 网络架构与布局
- **拓扑**: 输入层 (40节点) -> H1 (28) -> H2 (20) -> H3 (14) -> 输出层 (10)。
- **布局**: `NET_START_X = 420`。整体重心左移，为右侧 100.0% 概率标签预留安全空间。
- **标注**: 顶部指标汉化（训练轮数、损失函数、准确率）；层级标识避开状态栏。

## 4. 核心动画剧本 (Total: 45s)

### 阶段 1: 数据建模与特征提取 (0s - 8s)
- **动作**: 14x14 像素网格以“弹性生长”特效逐点构建数字 "5"。
- **视觉**: 每个像素点从 scale(0) 弹跳至 scale(1)，模拟原始传感数据的捕获过程。

### 阶段 2: 首次全训练环路 (8s - 22s)
- **前向传播 (FWD)**: 20px 宽的**亮青色**脉冲沿突触滑动（8s-18s）。
- **预测偏差**: 输出层展示“识别为 3”的混乱分布。
- **反向传播 (BWD)**: 25px 宽的**亮橘红色**能量波从输出层逆向回传至隐藏层 1（20s-22s），模拟权重的强行重写。

### 阶段 3: 深度迭代与收敛 (22s - 38s)
- **进阶学习**: 切换输入为数字 "8"。
- **视觉**: 线条变细但流动加快，Metrics 指标（Epoch/Loss/Acc）进入长达 15 秒的平滑滚动。
- **目标**: Loss 降至 0.0024 以下，准确率稳定在 99% 以上。

### 阶段 4: 推理验证与生产演示 (38s - 45s)
- **动作**: 切换全新样本 "7"。
- **推理**: 15px 白色脉冲瞬间贯穿网络。
- **结果**: Softmax 输出拟人化分布（"7" 占 92.4%，"1" 占 4.2%），并弹出右侧小型结果窗。

## 5. 录制与控制协议
- **全局控制**: `window.mnistDemo`
    - `step(t)`: 跳转至毫秒级时间戳。
    - `getDuration()`: 返回 45000。
- **字幕标准**: 字体 SimHei，加粗灰度，绝对避让左侧矩阵区域。
