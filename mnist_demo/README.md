# MNIST 神经网络训练可视化 (45秒增强版)

本项目是一个基于 Vue 3 和 Anime.js 构建的高质量神经网络训练过程可视化动画。通过 45 秒的精细刻画，生动展示了深度学习模型从原始数据加载、向量化、多次训练迭代（前向传播与反向传播）到最终推理验证的全过程。

## 动画剧本与字幕
1. **[00-08s] 数据准备**：加载并预处理 MNIST 原始手写图像数据集，构建 14x14 归一化输入张量。
2. **[08-22s] 首次训练**：前向传播计算预测分布，发现误差后启动反向传播，逆向修正每一层权重。
3. **[22-38s] 持续迭代**：加载数字 "8" 进行进阶迭代，权重矩阵根据误差历史进行自适应深度优化。
4. **[38-45s] 推理验证**：输入新样本 "7"，展示已训练模型如何基于特征空间做出毫秒级分类决策。

## 关键视觉特性
- **确定性信号流**：采用高对比度实线脉冲（FWD 20px / BWD 25px），清晰展示层间数据交换。
- **动态指标更新**：训练轮数、损失函数（Loss）与准确率随动画进度平滑滚动。
- **高保真点阵**：14x14 像素矩阵配合弹性缩放效果，完美还原 MNIST 特征提取过程。
- **极致布局**：针对 1080p 录制优化，全场黑体加粗字幕，元素互不遮挡。

## 使用与录制说明
开发预览：`npm run dev` | 生产构建：`npm run build && npm run preview` | 视频录制：`npm run record`

### 一键执行命令 (方便复制)
```bash
cd mnist_demo && npm install && npm run dev
```
```bash
cd mnist_demo && npm run record
```

## 录制标准
- **输出路径**: `video_records/mnist_demo.mp4`
- **规格**: 1080p / 60fps
- **控制接口**: `window.mnistDemo.step(timestamp)`
